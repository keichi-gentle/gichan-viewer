================================================================================
기찬뷰어 (Gichan Viewer) - 요구사항 정의서
================================================================================

프로젝트 개요: 기찬다이어리(WPF) 데이터를 스마트폰에서 조회할 수 있는 PWA 뷰어
상위 프로젝트: LifeRecord (기찬다이어리 WPF)
관계: 읽기 전용 동반앱 (Companion App)

================================================================================

1. 데이터 소스
   - 기찬다이어리 WPF에서 생성한 Excel 파일 (.xlsx)
   - 컬럼 구조: A~Q (17개 컬럼) — 기존 LifeRecord와 동일
   - 파일 가져오기: 사용자가 수동으로 Excel 파일 선택 (파일 선택기)
   - 가져온 데이터는 로컬에 캐시하여 오프라인에서도 조회 가능

2. 플랫폼
   - PWA (Progressive Web App)
   - 호스팅: GitHub Pages (무료)
   - 대상: 스마트폰/태블릿 브라우저 (Chrome, Safari)
   - 설치: "홈 화면에 추가"로 앱처럼 사용
   - 오프라인: Service Worker + IndexedDB로 지원

3. 현황탭 (대시보드)
   - 수유 카드: 최근 수유 시간, 경과시간, 세부내용(분할 N회 포함), 오늘 누적량/횟수
   - 배변 카드: 소변/대변 최근 시간, 경과시간, 오늘 횟수
   - 위생 카드: 세안/목욕/손발톱 최근 시간, 경과시간 (N일 N시간 경과)
   - 신체 카드: 키/몸무게/머리둘레 최근 기록
   - 건강/기타 카드: 최근 기록 표시
   - 카드 세로 1열 배치 (모바일 세로 화면 최적화)

4. 조회탭 (리스트)
   - 날짜 범위 필터 (시작~종료)
   - 카테고리 체크박스 6개 (수유/배변/위생/신체/건강/기타, 전체 기본 체크)
   - 키워드 검색
   - 리스트 표시: 날짜, 시간, 구분, 세부내용, 양
   - 세부내용 카테고리별 전경색 적용 (배변은 소변/대변 색 구분)
   - 구분(카테고리) 카테고리별 전경색 적용
   - 페이지네이션 (건수는 설정탭에서 조절)
   - 정렬: 최신순/오래된순 토글

5. 리포트탭 (차트)
   - 기간 선택: 7일 / 14일 / 30일 / 전체
   - 요약 카드 5개: 오늘 수유, 오늘 수유량, 오늘 배변, 평균 수유텀, 일평균 수유량
   - 차트 6종 (세로 배치):
     1. 1회 수유량 변화 추이 (Line)
     2. 일별 수유량 추이 (Line)
     3. 수유텀 분포 (Bar)
     4. 일별 배변 횟수 (Grouped Bar — 소변/대변 나란히)
     5. 카테고리별 이벤트 비율 (Doughnut)
     6. 일별 이벤트 현황 (Grouped Bar — 6개 카테고리)

6. 설정탭
   - Excel 파일 가져오기 버튼
   - 마지막 Import 정보 표시 (날짜, 건수)
   - 페이지당 표시 건수 선택 (10/20/30/50)
   - 아기 이름 (수동 입력, 로컬 저장)
   - 테마 전환 (Dark/Light)

7. 디자인
   - 색상 팔레트: WPF 기찬다이어리와 동일 (Set 10 로우라이트)
   - Dark/Light 테마 전환 지원
   - 모바일 세로 화면 최적화 레이아웃
   - 하단 탭바 네비게이션 (4탭)
   - 터치 친화적 UI (버튼/카드 크기)

8. 기술 스택
   - HTML5 + CSS3 + Vanilla JavaScript (프레임워크 없음, 빌드 도구 없음)
   - Chart.js — 차트 렌더링
   - SheetJS (xlsx) — Excel 파일 파싱
   - Service Worker — PWA 오프라인 캐시
   - IndexedDB — Import 데이터 로컬 캐시
   - LocalStorage — 설정값 저장

9. Excel 호환성
   - 기존 컬럼 A~Q 매핑 (LifeRecord WPF와 동일)
   - "위생관리","통증" 등 레거시 카테고리 읽기 지원
   - K~Q 컬럼 없는 기존 파일 → 빈값 처리

10. 비기능 요구사항
    - 앱스토어 등록 불필요 (PWA)
    - 서버 비용 0원 (GitHub Pages)
    - 오프라인 동작 (마지막 Import 데이터 캐시)
    - 로딩 속도 3초 이내 (최초 접속 기준)
